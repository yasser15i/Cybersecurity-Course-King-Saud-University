🔍 كيفية الحصول على Google Forms Entry IDs

الخطوة 1: افتح Google Form

اذهب إلى نموذجك واضغط F12 لفتح Developer Tools

الخطوة 2: افتح Network Tab

في Developer Tools، اذهب إلى تبويب Network

الخطوة 3: امسح Network Log

اضغط على أيقونة Clear (🗑️) لمسح السجلات

الخطوة 4: أرسل النموذج

املأ النموذج بأي بيانات تجريبية واضغط Submit

الخطوة 5: ابحث عن formResponse

في Network tab، ابحث عن request باسم formResponse

الخطوة 6: انسخ Entry IDs

اضغط على formResponse request، ثم اذهب إلى Form Data وانسخ entry IDs

⚠️ تحذير: Entry IDs تبدو مثل هذا: entry.1234567890 - تأكد من نسخها بالضبط
// مثال على Entry IDs الصحيحة: formDataObj.append('entry.1234567890', formData.full_name); // full_name formDataObj.append('entry.1234567891', formData.email); // email formDataObj.append('entry.1234567892', formData.phone); // phone formDataObj.append('entry.1234567893', formData.specialization); // specialization formDataObj.append('entry.1234567894', formData.experience_level); // experience_level formDataObj.append('entry.1234567895', formData.motivation); // motivation

الخطوة 7: حدث الكود

انسخ Entry IDs الصحيحة من Network tab والصقها في RegisterSection.tsx
